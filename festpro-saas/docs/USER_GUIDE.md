## 1. എന്താണ് FestPro?

കലാമത്സരങ്ങളും ഉത്സവങ്ങളും സംഘടിപ്പിക്കുന്നവർക്കായുള്ള ഒരു സമ്പൂർണ്ണ ഡിജിറ്റൽ പ്ലാറ്റ്ഫോമാണ് FestPro.

### ആരൊക്കെ ഉപയോഗിക്കും?

| റോൾ | എന്ത് ചെയ്യും |
|------|-------------|
| **അഡ്മിൻ** | ഫെസ്റ്റിവൽ സൃഷ്ടിക്കുക, ടീമിനെ മാനേജ് ചെയ്യുക |
| **ഫെസ്റ്റിവൽ ഡയറക്ടർ** | മത്സരങ്ങൾ, ഷെഡ്യൂൾ, ഫലങ്ങൾ എന്നിവ നിയന്ത്രിക്കുക |
| **വിധികർത്താവ് (Judge)** | പങ്കാളികൾക്ക് സ്കോർ നൽകുക |
| **പങ്കാളി** | രജിസ്റ്റർ ചെയ്യുക, ഷെഡ്യൂൾ കാണുക, ഫലം അറിയുക |
| **ഫിനാൻസ്** | ഫീസ്, ഇൻവോയ്സ്, പേയ്മെന്റ് കൈകാര്യം ചെയ്യുക |
| **വോളന്റിയർ** | ചെക്ക്-ഇൻ, ക്രൗഡ് മാനേജ്മെന്റ് |
| **മീഡിയ** | ഫോട്ടോ/വീഡിയോ അപ്ലോഡ് ചെയ്യുക |
| **റിസപ്ഷൻ** | ഓൺ-സൈറ്റ് രജിസ്ട്രേഷൻ നടത്തുക |

## 2. ആരംഭിക്കുന്നു

### സംഘാടകർക്കായി (Organization Admin)

**ഘട്ടം 1: അക്കൗണ്ട് ഉണ്ടാക്കുക**

1. **festpro.app**-ലേക്ക് പോകുക
2. **Sign Up** ക്ലിക്ക് ചെയ്യുക → ഇമെയിൽ + പാസ്വേഡ് നൽകുക
3. ഇമെയിൽ വെരിഫൈ ചെയ്യുക

**ഘട്ടം 2: ഓർഗനൈസേഷൻ സൃഷ്ടിക്കുക**

1. ലോഗിൻ ചെയ്ത ശേഷം **Create Organization** എന്നത് ക്ലിക്ക് ചെയ്യുക
2. നൽകേണ്ട വിവരങ്ങൾ:
   - **ഓർഗനൈസേഷന്റെ പേര്** (ഉദാ: "ജില്ലാ യുവജനോത്സവം")
   - **ലോഗോ** (ഐച്ഛികം)
   - **കോൺടാക്റ്റ് വിവരങ്ങൾ**
3. **Create** ക്ലിക്ക് ചെയ്യുക

**ഘട്ടം 3: ടീം അംഗങ്ങളെ ക്ഷണിക്കുക**

1. **Settings → Team** എന്നതിലേക്ക് പോകുക
2. **Invite Member** ക്ലിക്ക് ചെയ്യുക
3. ഇമെയിൽ നൽകി റോൾ (Admin/Manager/Finance/Media) തിരഞ്ഞെടുക്കുക
4. അവർക്ക് ഇൻവൈറ്റ് ഇമെയിൽ ലഭിക്കും

### പങ്കാളികൾക്കായി

1. സംഘാടകർ നൽകുന്ന **Public Portal** ലിങ്ക് തുറക്കുക
2. ഫെസ്റ്റിവലും മത്സരങ്ങളും ബ്രൗസ് ചെയ്യുക
3. **Register** ക്ലിക്ക് ചെയ്ത് വിവരങ്ങൾ നൽകുക
4. ആവശ്യമെങ്കിൽ ഫീസ് അടയ്ക്കുക
5. നിങ്ങളുടെ ഡാഷ്ബോർഡിൽ നിന്ന് ഷെഡ്യൂളും ഫലങ്ങളും കാണുക

## 3. റോളുകൾ വിശദീകരണം

### Organization Admin
- എല്ലാത്തിനും പൂർണ്ണ പ്രവേശനം
- ടീം അംഗങ്ങളെ ചേർക്കാം / നീക്കം ചെയ്യാം
- ബില്ലിംഗ്, സെറ്റിംഗ്സ് എന്നിവ കൈകാര്യം ചെയ്യാം

### Festival Director
- ഫെസ്റ്റിവലുകൾ സൃഷ്ടിക്കാം
- വിധികർത്താക്കളെ നിയോഗിക്കാം
- ഫലങ്ങൾ പ്രസിദ്ധീകരിക്കാം

### Manager
- ദൈനംദിന പ്രവർത്തനങ്ങൾ
- ഷെഡ്യൂൾ, വോളന്റിയർമാർ എന്നിവ കൈകാര്യം ചെയ്യാം

### Staff
- ലോജിസ്റ്റിക്സ് സഹായം
- പങ്കാളി വിവരങ്ങൾ അപ്ഡേറ്റ് ചെയ്യാം

### Judge (വിധികർത്താവ്)
- തനിക്ക് നിയോഗിച്ച മത്സരങ്ങൾ മാത്രം കാണാം
- സ്കോറും അഭിപ്രായങ്ങളും നൽകാം

### Finance
- ഇൻവോയ്സ്, പേയ്മെന്റ്, റീഫണ്ട്
- സാമ്പത്തിക റിപ്പോർട്ടുകൾ

### Media (മീഡിയ)
- ഫോട്ടോ, വീഡിയോ അപ്ലോഡ്
- ഗാലറി മാനേജ്മെന്റ്

### Reception (റിസപ്ഷൻ)
- ഓൺ-സൈറ്റ് ചെക്ക്-ഇൻ
- വാക്ക്-ഇൻ രജിസ്ട്രേഷൻ

### Volunteer (വോളന്റിയർ)
- അസൈൻ ചെയ്ത ജോലികൾ കാണുക
- ഹാജർ രേഖപ്പെടുത്തുക

## 4. ഫെസ്റ്റിവൽ സൃഷ്ടിക്കൽ

### ഘട്ടം ഘട്ടമായുള്ള പ്രക്രിയ

1. **ഡാഷ്ബോർഡിൽ** നിന്ന് **Create Festival** ക്ലിക്ക് ചെയ്യുക
2. വിവരങ്ങൾ പൂരിപ്പിക്കുക:

```
ഫെസ്റ്റിവലിന്റെ പേര് | "ജില്ലാ യുവജനോത്സവം 2025"
വിവരണം              | ഇവന്റിനെക്കുറിച്ച് ചുരുക്കം
ആരംഭ തീയതി         | 15-01-2025
അവസാന തീയതി         | 20-01-2025
വേദി                | മെയിൻ ഓഡിറ്റോറിയം
പരമാവധി പങ്കാളികൾ   | 500
രജിസ്ട്രേഷൻ ഫീസ്    | ₹100 (അല്ലെങ്കിൽ സൗജന്യം)
സ്റ്റാറ്റസ്          | ഡ്രാഫ്റ്റ് / പ്രസിദ്ധീകരിച്ചത്
```

3. **വേദികൾ** ചേർക്കുക:
   - മെയിൻ ഹാൾ (500 പേർ)
   - മിനി തിയേറ്റർ (100 പേർ)

4. **Save** ക്ലിക്ക് ചെയ്യുക

### ഫെസ്റ്റിവൽ ക്രമീകരണങ്ങൾ

- **Publish** — പബ്ലിക് പോർട്ടലിൽ കാണിക്കും
- **Unpublish** — മറയ്ക്കും
- **Close Registrations** — പുതിയ രജിസ്ട്രേഷൻ നിർത്തും
- **Cancel** — എല്ലാവർക്കും അറിയിപ്പ് നൽകി റദ്ദാക്കും

## 5. മത്സരങ്ങൾ കൈകാര്യം ചെയ്യുക

### മത്സരം സൃഷ്ടിക്കുക

1. ഫെസ്റ്റിവൽ തുറക്കുക → **Competitions** ടാബ്
2. **Add Competition** ക്ലിക്ക് ചെയ്യുക
3. വിവരങ്ങൾ നൽകുക:

```
പേര്          | "കഥകളി" / "ഗ്രൂപ്പ് ഗാനം"
വിഭാഗം        | നൃത്തം / സംഗീതം / നാടകം / കല
പ്രായപരിധി    | സബ്-ജൂനിയർ / ജൂനിയർ / സീനിയർ / ഓപ്പൺ
പരമാവധി എൻട്രി | 30
ദൈർഘ്യം       | 10 മിനിറ്റ്
സ്കോറിംഗ് രീതി | പോയിന്റ് അടിസ്ഥാനത്തിൽ / റാങ്ക് അടിസ്ഥാനത്തിൽ
```

4. **സ്കോറിംഗ് മാനദണ്ഡങ്ങൾ** സെറ്റ് ചെയ്യുക (ഉദാ: ടെക്നിക് 30%, പ്രസന്റേഷൻ 25%)
5. **വിധികർത്താക്കളെ** അസൈൻ ചെയ്യുക

### ഷെഡ്യൂൾ ചെയ്യുക

1. **Schedule** ടാബിലേക്ക് പോകുക
2. മത്സരങ്ങൾ ടൈംലൈനിൽ വയ്ക്കുക
3. വേദിയും സമയവും നൽകുക
4. സിസ്റ്റം ക്ലാഷ് ചെക്ക് ചെയ്യും
5. പ്രസിദ്ധീകരിക്കുക — പങ്കാളികൾക്ക് കാണാം

### മത്സരത്തിന്റെ അവസ്ഥകൾ

```
ഡ്രാഫ്റ്റ് → രജിസ്ട്രേഷൻ തുറന്നു → രജിസ്ട്രേഷൻ അവസാനിച്ചു →
വിധി നടക്കുന്നു → ഫലം പ്രസിദ്ധീകരിച്ചു → സർട്ടിഫിക്കറ്റ് തയ്യാറായി
```

## 6. രജിസ്ട്രേഷൻ & പങ്കാളികൾ

### പങ്കാളി രജിസ്ട്രേഷൻ (സ്വയം)

1. പബ്ലിക് പോർട്ടൽ സന്ദർശിക്കുക
2. ഫെസ്റ്റിവൽ തിരഞ്ഞെടുക്കുക
3. പേര്, ഫോൺ, ഇമെയിൽ, പ്രായം എന്നിവ നൽകുക
4. മത്സര വിഭാഗങ്ങൾ തിരഞ്ഞെടുക്കുക
5. ഫീസ് അടയ്ക്കുക (UPI/കാർഡ്/ഓഫ്ലൈൻ)
6. കൺഫർമേഷൻ ലഭിക്കും
7. ഡാഷ്ബോർഡിൽ നിന്ന് ഷെഡ്യൂളും ഹാൾ പാസും കാണുക

### പങ്കാളി രജിസ്ട്രേഷൻ (സംഘാടകർ)

1. **Participants** ടാബിലേക്ക് പോകുക
2. **Add Participant** ക്ലിക്ക് ചെയ്യുക
3. വിവരങ്ങൾ നൽകി മത്സരം തിരഞ്ഞെടുക്കുക
4. Save ചെയ്യുക

### ബൾക്ക് ഇറക്കുമതി

CSV/Excel ഫയൽ വഴി ഒരുമിച്ച് നിരവധി പങ്കാളികളെ ചേർക്കാം.

### രജിസ്ട്രേഷൻ സ്റ്റാറ്റസുകൾ

| സ്റ്റാറ്റസ് | വിവരണം |
|------------|---------|
| Pending | പേയ്മെന്റ് നടന്നിട്ടില്ല |
| Confirmed | രജിസ്റ്റർ ചെയ്തു, പേയ്മെന്റ് ആയി |
| Withdrawn | പങ്കാളി പിൻമാറി |
| Disqualified | സംഘാടകർ ഒഴിവാക്കി |
| Attended | ഇവന്റ് ദിവസം ഹാജരായി |

## 7. വിധികർത്താക്കൾ & സ്കോറിംഗ്

### വിധികർത്താക്കൾക്കുള്ള ഗൈഡ്

1. **ലോഗിൻ** ചെയ്യുക
2. **അസൈൻ ചെയ്ത മത്സരങ്ങൾ** കാണാം
3. മത്സരത്തിൽ ക്ലിക്ക് ചെയ്യുക → പങ്കാളികളുടെ ലിസ്റ്റ്
4. ഓരോ പങ്കാളിക്കും സ്കോർ നൽകുക:

```
പങ്കാളി: അരവിന്ദ് കെ.
ക്രൈറ്റീരിയ 1 (ടെക്നിക് 30%):  [ 28/30 ]
ക്രൈറ്റീരിയ 2 (പ്രസന്റേഷൻ 25%): [ 22/25 ]
ക്രൈറ്റീരിയ 3 (ഉള്ളടക്കം 25%):   [ 23/25 ]
ക്രൈറ്റീരിയ 4 (ഓവറോൾ 20%):     [ 18/20 ]
ആകെ: 91/100
കുറിപ്പ്: മികച്ച പ്രകടനം
```

5. സ്കോർ സമർപ്പിക്കുക — ഫലം പ്രസിദ്ധീകരിക്കും വരെ തിരുത്താം

### സ്കോറിംഗ് രീതികൾ

- **പോയിന്റ് അടിസ്ഥാനത്തിൽ**: ഓരോ ക്രൈറ്റീരിയയ്ക്കും ഭാരം നിശ്ചയിക്കും
- **റാങ്ക് അടിസ്ഥാനത്തിൽ**: 1st, 2nd, 3rd എന്നിങ്ങനെ റാങ്ക് നൽകും
- **Pass/Fail**: യോഗ്യതാ റൗണ്ടുകൾക്ക്

### ഫല പ്രക്രിയ

1. **സ്കോർ അന്തിമമാക്കുക** — വിധികർത്താക്കളുടെ എൻട്രി ലോക്ക് ചെയ്യുക
2. **റാങ്ക് ഓട്ടോ-കണക്കുകൂട്ടുക** — സിസ്റ്റം വിജയികളെ നിർണ്ണയിക്കും
3. **ഫലം പ്രിവ്യൂ ചെയ്യുക** — അഡ്മിൻ പരിശോധിക്കും
4. **പ്രസിദ്ധീകരിക്കുക** — പങ്കാളികൾക്കും പൊതുജനങ്ങൾക്കും കാണാം
5. **സർട്ടിഫിക്കറ്റ്** — വിജയികൾക്ക് ഓട്ടോ-ജനറേറ്റ് ചെയ്യും

## 8. ഫലങ്ങൾ & സർട്ടിഫിക്കറ്റുകൾ

### ഫലങ്ങൾ കാണുക

**പങ്കാളിയായി:**
- **My Results** → സ്കോർ, റാങ്ക്, കുറിപ്പുകൾ കാണാം
- പങ്കാളിത്ത സർട്ടിഫിക്കറ്റ് ഡൗൺലോഡ് ചെയ്യാം

**സംഘാടകരായി:**
- **Results Dashboard** — എല്ലാ പ്രസിദ്ധീകരിച്ച ഫലങ്ങളും
- PDF/Excel ആയി എക്സ്പോർട്ട് ചെയ്യാം

### സർട്ടിഫിക്കറ്റുകൾ

| തരം | എപ്പോൾ |
|------|---------|
| പങ്കാളിത്ത സർട്ടിഫിക്കറ്റ് | മത്സരം കഴിഞ്ഞയുടൻ |
| വിജയി (1st/2nd/3rd) | ഫലം പ്രസിദ്ധീകരിച്ച ശേഷം |
| അഭിനന്ദന സർട്ടിഫിക്കറ്റ് | പ്രത്യേക അംഗീകാരത്തിന് |
| വിധികർത്താവ് | നന്ദി സൂചകമായി |
| വോളന്റിയർ | സേവനത്തിനുള്ള അംഗീകാരം |

**ഇഷ്ടാനുസൃതമാക്കാം:**
- ലോഗോ, ഒപ്പ്, ബാക്ക്‌ഗ്രൗണ്ട് എന്നിവ ചേർക്കാം
- ഓട്ടോ-ഫിൽ: പേര്, റാങ്ക്, മത്സരം, തീയതി
- PDF ആയി ഡൗൺലോഡ് (ഒറ്റ / ബൾക്ക്)
- ഇമെയിൽ / WhatsApp വഴി ഷെയർ ചെയ്യാം

- ചെലവുകൾ (വേദി, ഉപകരണങ്ങൾ, കാറ്ററിംഗ്)
- അറ്റാദായം/നഷ്ടം

## 10. വോളന്റിയർമാർ & ഓപ്പറേഷൻസ്

### വോളന്റിയർ മാനേജ്മെന്റ്

1. **വോളന്റിയർമാരെ ചേർക്കുക** → പേര്, ഫോൺ, കഴിവുകൾ
2. **ഷിഫ്റ്റുകൾ സൃഷ്ടിക്കുക** → തീയതി, സമയം, സ്ഥലം
3. **ജോലികൾ അസൈൻ ചെയ്യുക** → ചെക്ക്-ഇൻ ഡെസ്ക്, സ്റ്റേജ് മാനേജ്മെന്റ്
4. **ഹാജർ രേഖപ്പെടുത്തുക** → QR സ്കാൻ / മാനുവൽ

### ഓൺ-സൈറ്റ് ഓപ്പറേഷൻസ്

- **ചെക്ക്-ഇൻ ഡെസ്ക്:** QR വഴി പങ്കാളി വെരിഫിക്കേഷൻ
- **സ്റ്റേജ് മാനേജ്മെന്റ്:** തത്സമയ മത്സര നില
- **അറിയിപ്പുകൾ:** ആപ്പ്/SMS വഴി
- **മെഡിക്കൽ:** പ്രഥമശുശ്രൂഷ, സംഭവ റിപ്പോർട്ടിംഗ്

## 13. പൊതുവായ ചോദ്യങ്ങൾ

### പൊതു ചോദ്യങ്ങൾ

**ചോദ്യം: പാസ്വേഡ് എങ്ങനെ റീസെറ്റ് ചെയ്യും?**
ഉത്തരം: **Forgot Password** ക്ലിക്ക് ചെയ്യുക → ഇമെയിൽ ചെക്ക് ചെയ്യുക.

**ചോദ്യം: ഫോണിൽ FestPro ഉപയോഗിക്കാമോ?**
ഉത്തരം: അതെ. PWA ആയി ഇൻസ്റ്റാൾ ചെയ്യാം.

**ചോദ്യം: FestPro സൗജന്യമാണോ?**
ഉത്തരം: ചെറിയ ഇവന്റുകൾക്ക് സൗജന്യ പ്ലാൻ ലഭ്യമാണ്.

### സംഘാടകർക്ക്

**ചോദ്യം: ഒന്നിൽ കൂടുതൽ ഫെസ്റ്റിവലുകൾ നടത്താമോ?**
ഉത്തരം: അതെ. എത്ര വേണമെങ്കിലും സൃഷ്ടിക്കാം.

**ചോദ്യം: ടീമിന് ഒരുമിച്ച് ജോലി ചെയ്യാമോ?**
ഉത്തരം: അതെ. വ്യത്യസ്ത റോളുകളിൽ ടീം അംഗങ്ങളെ ചേർക്കാം.

**ചോദ്യം: ഡാറ്റ എക്സ്പോർട്ട് ചെയ്യാമോ?**
ഉത്തരം: അതെ. CSV/PDF ഫോർമാറ്റിൽ.

**ചോദ്യം: രജിസ്ട്രേഷൻ റദ്ദാക്കാമോ?**
